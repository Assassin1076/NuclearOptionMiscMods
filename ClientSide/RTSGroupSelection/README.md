仿照RTS类游戏设计的组选择器，你可以：

1. 按下上方数字键0-9来快速锁定预设的组（抛弃当前锁定的目标）
2. 按下LCtrl+[0-9]保存当前锁定的组到预设
3. 按下LShift+[0-9]附加预设的组到当前锁定
   
所有快捷键都是可自定义的

A group selector designed in the style of RTS games. You can:

1. Press the number keys 0-9 to quickly lock onto preset groups (releasing the currently locked targets).
2. Press LCtrl + [0-9] to save the currently locked group to a preset.
3. Press LShift + [0-9] to add a preset group to the currently locked targets.

All hotkeys are fully customizable.
