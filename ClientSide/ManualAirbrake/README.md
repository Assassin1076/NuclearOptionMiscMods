将你的减速板绑定在刹车按键上。

现在将节流阀收到最小不会自动打开减速板，你需要手动按下刹车键来打开减速板

对KR-67、SFB-81这类使用开裂翼的飞行器无效！

Bind your airbrakes to the brake key.

Now pulling the throttle to minimum will no longer automatically deploy the airbrakes.
You must manually press the brake key to deploy the airbrakes.

Does not work for aircraft that use split wings, such as the KR-67 and SFB-81!
